Create `.github/workflows/ci.yml` (create `.github/workflows/` directory first).

Per D-08: Three parallel jobs — `fmt`, `clippy`, `test`. Each runs independently.
Per D-09: Use `dtolnay/rust-toolchain` + `Swatinem/rust-cache`. Rust stable channel. Do NOT use `actions-rs/*`.
Per D-10: Pin ALL third-party actions to full commit SHA. Add `# vN.N.N` comment after each SHA for readability.

Workflow structure:
```yaml
name: CI
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

env:
  CARGO_TERM_COLOR: always

jobs:
  fmt:
    name: Format
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@{CHECKOUT_SHA}  # v4
      - uses: dtolnay/rust-toolchain@{TOOLCHAIN_SHA}  # stable
        with:
          toolchain: stable
          components: rustfmt
      - run: cargo fmt --all -- --check

  clippy:
    name: Clippy
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@{CHECKOUT_SHA}  # v4
      - uses: dtolnay/rust-toolchain@{TOOLCHAIN_SHA}  # stable
        with:
          toolchain: stable
          components: clippy
      - uses: Swatinem/rust-cache@{CACHE_SHA}  # v2
      - run: cargo clippy --all-targets -- -D warnings

  test:
    name: Test
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@{CHECKOUT_SHA}  # v4
      - uses: dtolnay/rust-toolchain@{TOOLCHAIN_SHA}  # stable
        with:
          toolchain: stable
      - uses: Swatinem/rust-cache@{CACHE_SHA}  # v2
      - run: cargo test --workspace
```

Replace `{CHECKOUT_SHA}`, `{TOOLCHAIN_SHA}`, `{CACHE_SHA}` with the actual 40-char SHAs resolved in Task 1.
