       a. Define `AppState`:
          ```rust
          pub struct AppState {
              pub callback_tx: tokio::sync::mpsc::Sender<RawCallbackEvent>,
              pub nonce_map: HashMap<String, NonceMeta>,
              pub crawler_catalog: CrawlerCatalog,
          }
          ```
          Where `NonceMeta` is a small struct: `{ tier: u8, payload_id: String, embedding_loc: String }` parsed from callback-map.json at startup.

       b. Define callback handler (D-03: always returns 204):
          ```rust
          pub async fn callback_handler(
              Path(nonce): Path<String>,
              State(state): State<Arc<AppState>>,
              ConnectInfo(peer_addr): ConnectInfo<SocketAddr>,
              headers: HeaderMap,
          ) -> StatusCode {
              // Validate nonce: 16 lowercase hex chars
              if nonce.len() != 16 || !nonce.chars().all(|c| c.is_ascii_hexdigit() && !c.is_uppercase()) {
                  return StatusCode::NO_CONTENT;  // D-03
              }
              // Look up nonce
              let Some(meta) = state.nonce_map.get(&nonce) else {
                  return StatusCode::NO_CONTENT;  // D-03
              };
              // Extract fingerprint (pure function)
              let fingerprint = crate::fingerprint::extract(peer_addr.ip(), &headers);
              // Classify
              let classification = state.crawler_catalog.classify(&fingerprint.user_agent);
              // Assemble and send (non-blocking)
              let event = RawCallbackEvent {
                  nonce, tier: meta.tier, payload_id: meta.payload_id.clone(),
                  embedding_loc: meta.embedding_loc.clone(), fingerprint, classification,
                  received_at: now_unix_secs(),
              };
              let _ = state.callback_tx.try_send(event);
              StatusCode::NO_CONTENT  // D-03
          }
          ```
          CRITICAL: Do NOT extract Body, Bytes, String, Json, or Form from the request. Only Path, State, ConnectInfo, HeaderMap. This enforces SRV-07.
          CRITICAL: Use `ConnectInfo(peer_addr): ConnectInfo<SocketAddr>` and pass `peer_addr.ip()` to fingerprint extract. Do NOT use axum-client-ip in the handler directly (simplify -- ConnectInfo gives the direct peer IP; proxy-aware IP can be added later if needed). This avoids Pitfall 5 complexity.

       c. Define `pub async fn serve(config: &Config, project_path: &Path, json_mode: bool) -> anyhow::Result<()>`:
          - Load callback-map.json from `{project_path}/output/callback-map.json`. Deserialize as `Vec<NonceMapping>` (from types.rs). Build `HashMap<String, NonceMeta>` keyed by nonce.
          - Load crawler catalog via `CrawlerCatalog::load()?`

          - Create channels: `let (callback_tx, callback_rx) = tokio::sync::mpsc::channel(256);`
          - Create broadcast: `let (event_tx, _) = tokio::sync::broadcast::channel(1024);`
          - Subscribe consumers: `let db_rx = event_tx.subscribe(); let log_rx = event_tx.subscribe();`
          - Spawn broker: `tokio::spawn(broker::broker_task(callback_rx, event_tx));`
          - Spawn DB writer: `tokio::spawn(broker::db_writer_task(db_rx, conn.clone()));`
          - Spawn stdout logger: `tokio::spawn(broker::stdout_logger_task(log_rx, json_mode));`
          - Build router:

            let app = Router::new()

            ```
            Note: Axum 0.8 uses `{nonce}` not `:nonce` for path params.

            ```

            let listener = tokio::net::TcpListener::bind(&config.bind_address).await?;

                .with_graceful_shutdown(shutdown_signal())

            ```
            CRITICAL: Use `into_make_service_with_connect_info::<SocketAddr>()` NOT `into_make_service()` (Pitfall 1).

            ```
            let detections = conn.call(|conn| Ok(crate::store::count_detections(conn)?)).await?;
            println!("\nShutdown complete. {} detection(s) recorded.", detections);
            ```

       d. Define `async fn shutdown_signal()`:

          ```
