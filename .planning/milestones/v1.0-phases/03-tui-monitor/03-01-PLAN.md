    **AppState struct (per Research Pattern 2, UI-SPEC):**
    ```rust
    pub struct AppState {
        pub events: Vec<AppEvent>,
        pub filter: TierFilter,
        pub sort: SortField,
        pub show_replays: bool,        // D-05: false by default
        pub table_state: ratatui::widgets::TableState,
        pub at_bottom: bool,           // auto-scroll tracking
        pub new_events_count: usize,   // "N new events" indicator
        pub mode: UiMode,
        pub command_input: String,
        pub command_error: Option<(String, std::time::Instant)>,
        pub status_line: String,       // "Serving on ..." or "Attached to db: ..."
    }
    ```

    **AppState::new()** — initialize with filter=All, sort=Time, show_replays=false, at_bottom=true, mode=Normal, empty events vec.

    **AppState::push_event(&mut self, event: AppEvent):**
    - Append to self.events
    - If self.at_bottom, increment nothing (auto-scroll will handle)
    - If NOT at_bottom, increment self.new_events_count

    **AppState::visible_events(&self) -> Vec<&AppEvent>:**
    1. Start with self.events.iter()
    2. Filter: if show_replays==false, exclude events where is_replay==true
    3. Filter: match self.filter — All passes everything; T1 keeps tier==1; T2 keeps tier==2; T3 keeps tier==3
    4. Collect into Vec<&AppEvent>
    5. Sort based on self.sort:
       - Time: sort by received_at descending (newest first)
       - Tier: sort by tier ascending, then received_at descending as tiebreaker
       - Source: sort by fingerprint.source_ip.to_string() ascending, then received_at descending
    6. Return sorted vec

    **AppState::detection_count(&self) -> usize:**
    Count events where is_replay==false. This is the "Detections" header stat (D-07).

    **AppState::session_count(&self) -> usize:**
    Count distinct session_id values among events where is_replay==false.

    **AppState::tier_counts(&self) -> (usize, usize, usize):**
    Count events where is_replay==false, grouped by tier: (t1_count, t2_count, t3_count).

    **AppState::replay_count(&self) -> usize:**
    Count events where is_replay==true. Displayed as "N replays hidden" in header (D-06).

    **AppState::cycle_filter(&mut self):**
    Set self.filter = self.filter.next(). Reset table_state to select(Some(0)).

    **AppState::cycle_sort(&mut self):**
    Set self.sort = self.sort.next().

    **AppState::toggle_replays(&mut self):**
    Flip self.show_replays. Reset table_state to select(Some(0)) per Pitfall 4.
